# NeDarlin Data Science Project

This project is initialized with `uv` and common data science dependencies.

## Quick start

- Create/activate environment and install dependencies:
  - `uv sync`
- Run Jupyter:
  - `uv run jupyter lab`

## Structure

- `Data/` raw datasets
- `Experiments/` hypotheses, code, and results by experiment
- `notebooks/` exploratory notebooks
- `src/` reusable code
